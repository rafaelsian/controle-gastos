Como configurar:
1. Acesse github.com/settings/tokens → Generate new token (classic)
2. Marque apenas o escopo gist → Generate
3. Cole o token abaixo. Na primeira vez, deixe o Gist ID vazio e clique "Salvar no Gist" — ele criará um Gist privado automaticamente.
